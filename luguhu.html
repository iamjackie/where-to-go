成都出发 · 纯电SUV床车 · 周五晚出发

Plan B：直奔泸沽湖·在湖边待2天(3天标准版)

三天全晴！泸沽湖海拔2690m，4月月雨天仅~8天。白天23°短袖+外套，早晚5°厚外套（纯电开空调过夜无压力）。紫外线极强：高原+水面反射=双倍暴晒，SPF50+防晒+墨镜+帽子必备。

🔌 充电规划（本路线最关键）

⚠️ 西昌→泸沽湖250km全是山路(G348/S307)，800+弯道，限速40-60km/h，山路电耗比高速高30-50%。250km山路≈350-400km高速电耗。必须在西昌和盐源两个点充满！

全程充电节点一览

成都→西昌(440km高速)：荥经服务区(国网8桩60kW)补电 → 直达西昌(市区桩密集)
西昌→泸沽湖(250km山路)：西昌充满100%出发 → 盐源县城(170km处·国网/特来电快充·关键中继) → 泸沽湖景区(蔚来换电站+景区充电站3快充6枪)
返程同理反向充电。全程约需6-7次充电，每次20-30min。
⚡ 出发前在高德/特来电APP截图保存充电桩位置（山区信号弱）

🗺 泸沽湖环湖8大必去景点

环湖一圈68km·路况好(铺装路面)·自驾边走边停约4-5h。

❶ 四川情人滩 ⭐

免费20-30min有路边停车

白色细沙+渐变蓝绿湖水，泸沽湖四川侧最美沙滩。赤脚踩水拍照极出片。上午顺光最佳。

❷ 格姆女神山 ⭐（可坐索道）

索道90元/人索道+山顶1.5-2h

摩梭神山海拔3754m。索道到山腰女神洞+"天空之镜"观景台(2025新增·360°俯瞰泸沽湖全景)。时间紧可跳过，把时间留给里格和日落。

❸ 里格半岛 ⭐⭐⭐ 泸沽湖第一名片！

免费30-60min+村口停车场10元

三面环水的小半岛，摩梭木屋+精品客栈。里格观景台俯拍=所有宣传照的机位。走下去逛半岛+湖边发呆+咖啡馆。日落时半岛被金光笼罩最美。住宿首选区(清明300-600元/晚)。

❹ 大落水村 · 篝火晚会

篝火30元/人晚会1-1.5h·20:00开始

最大村落，吃住选择最多。晚上篝火晚会围火跳锅庄舞，游客可加入。气氛好，推荐体验。

❺ 草海 + 走婚桥 ⭐⭐⭐ 最诗意之地

免费30-60min有停车场·有卫生间

长满水草的浅水区+300m木栈桥。摩梭"走婚"文化象征。日落时草海金黄+栈桥剪影=美到窒息。旁有卫生间/商店，是床车过夜好选择。

❻ 女神湾 ⭐⭐⭐ 日落第一位！

免费日落前40min到有停车位

湖西岸，正对格姆女神山。泸沽湖看日落公认NO.1——湖面蓝→金→橘红→玫瑰紫，约40min。4月日落约18:30-19:00，17:30到占位。安静无商业。床车过夜绝佳(无光污染·银河!)。

❼ 猪槽船 ⭐⭐ 必体验

拼船50-80元/人30-40min

摩梭传统独木舟。湖水见底(能见度12m+)，船如悬浮。船夫唱摩梭情歌。⚠️ 正规码头买票，不找路边拉客的。清晨有晨雾时最梦幻。

❽ 泸沽湖观景台（景区入口前）

免费(景区外)15-20min

进景区前的全景俯瞰点。第一次看到泸沽湖全貌的地方——碧蓝湖面+雪山+湖中岛屿，震撼到说不出话。必停。

📸 拍照光线：上午东岸顺光(情人滩·女神山) → 下午/傍晚西岸日落(女神湾·草海走婚桥)。安排好方向出最好的片。

📋 行程方案

Plan A：西昌+泸沽湖都玩4天·最从容
Plan B：直奔泸沽湖3天·标准版
Plan A 适合：4/7周二不上班（或能请1天假）。在西昌玩半天+泸沽湖玩1.5天+从容返程。最推荐的方案！
Day 0 · 4/3 周五晚 — 成都→石棉过夜（3h·轻松）
🚗 20:00 出发 → 荥经补电(20min) → 23:00 石棉服务区过夜
仅开3h到石棉，23点舒服睡觉。服务区有卫生间/小卖部/充电桩。边睡边充满电。
Day 1 · 4/4 周六 — 石棉→西昌(2h)→邛海半日 ☀️ 27°
😴 08:30 自然醒 → 09:00 出发
石棉→西昌仅180km/2h。白天走雅西高速能看到拖乌山双螺旋隧道和干海子大桥的壮观工程。11:00到西昌。
🍽 11:30 邛海醉虾午餐 → 🚲 13:00 邛海骑行3h → 🔥 18:00 火盆烧烤
半天精华版：醉虾(人均60) → 环湖骑行(最美段梦里水乡+烟雨鹭洲约17km/3h·租车90元) → 火盆烧烤(藿香/黑哥/胡四哥·2人120-180元)。详见西昌攻略。
🌙 20:30 西昌过夜·充满电（明天满电出发去泸沽湖！）
观海路营地或万达停车场(有桩)。务必充至100%。
Day 2 · 4/5 周日 — 西昌→泸沽湖(5h)+半环湖 ☀️ 24°
🚗 07:30 满电出发·西昌→泸沽湖
250km山路/4-5h。800+弯道，慢行安全第一。沿途翻磨盘山(1700m)+小高山(3200m)，风景从河谷到高原草甸变化极大。
⚡🍎 10:00 盐源县城补电(20-30min)+买盐源苹果
170km处。国网/特来电快充。盐源苹果3-5元/斤又脆又甜。
📸 12:00 泸沽湖观景台(震撼全景!) → 🎫 12:15 买票入景区(70元/人)
翻过最后一座山·第一眼看到泸沽湖·碧蓝到不真实。观景台必停拍照。
⚡🍽 12:30 大落水村 充电+午餐
到达后第一件事找充电桩（景区充电站3快充6枪/蔚来换电站）。插上充电去吃饭。摩梭猪膘肉+荞麦粑粑+苏里玛酒，人均40-60元。
🚗 14:00-18:30 半环湖(东线精华)
情人滩(拍照20min) → 格姆女神山(索道可选1.5h) → 里格半岛(发呆拍照40min+) → 驱车到女神湾看日落🌅(17:30到·今天高潮！)
🍽 19:30 回大落水晚餐 → 🔥 20:00 篝火晚会(30元·可选)
围火跳舞，气氛好。
🌙 22:00 湖边床车过夜·看银河
过夜首选：女神湾停车场(极安静·面湖·无光污染·银河肉眼可见！但无卫生间)。备选：草海停车场(有卫生间/商店)、大落水停车场(最全配套)。
Day 3 · 4/6 周一 — 环湖下半圈+返程到西昌 ☀️ 23°
🌅 06:50 泸沽湖日出 → 08:30 环湖西线
日出在情人滩/大落水码头看。然后走婚桥+草海(晨光最美)+猪槽船(拼船60元/人·30min)。
🍽 11:00 午餐 → ⚡充满电 → 12:00 出发返回西昌
泸沽湖→盐源补电→西昌。约4-5h。到西昌~17:00。
🔥 17:30 西昌告别晚餐(火盆烧烤/醉虾) + ⚡充电
在西昌吃晚饭+充满电。可以在西昌再住一晚（第二天轻松回），或者当晚开到石棉过夜(180km/2h)。
Day 4 · 4/7 周二 — 从容返程·走走停停
😴 自然醒 → 沿途停留安顺场(红军渡·免费·1h)+汉源湖(小泸沽湖·免费·20min)
不赶路，走走停停回成都。下午到家。详细沿途景点见西昌攻略返程部分。
Plan B 适合：只有4/4-6三天假期，想把时间全给泸沽湖。西昌仅作为充电中转。节奏较紧但可行。
Day 0 · 4/3 周五晚 — 成都→石棉过夜（3h·轻松）
同Plan A · 20:00出发 → 23:00石棉服务区过夜+充电
3h轻松到达，不用肝到凌晨。
Day 1 · 4/4 周六 — 石棉→西昌充电→泸沽湖(全天赶路+半环湖)
🚗 07:00 出发 · 石棉→西昌(2h)→快充30min→泸沽湖(4-5h)
今天是赶路日。石棉→西昌180km/2h → 西昌快充至100%(30min·趁机买早餐/上厕所) → 西昌→盐源(170km·补电) → 泸沽湖。约13:00到达泸沽湖。
⚡🍽 13:00 到达·充电+午餐 → 14:00-18:30 半环湖(东线精华)
同Plan A的Day2下午行程：情人滩→里格半岛→女神湾日落🌅。晚上篝火晚会(可选)。女神湾过夜看银河。
Day 2 · 4/5 周日 — 环湖完整版+深度体验 ☀️ 24°
🌅 06:50 日出 → 全天环湖+猪槽船+发呆
今天是核心游玩日。日出→环湖西线(走婚桥+草海)→猪槽船(60元·30min)→午餐→下午里格发呆/骑行/达祖村→女神湾二刷日落→晚餐。
不赶，慢慢来。泸沽湖的精髓是慢下来。
🌙 过夜 + ⚡ 充满电（明天返程700km！）
今晚务必找充电桩充满！明天全天赶路。
Day 3 · 4/6 周一 — 返程700km（全天赶路·走走停停缓解疲劳）
今天返程700km需8-9h（含充电吃饭）。07:00出发，争取19:00到家。建议两人轮换驾驶。
🌅 06:30 最后一个日出 → 07:00 出发
泸沽湖→盐源补电(20min)→西昌充电+午饭(1h)→雅西高速(荥经补电)→成都。
🍽 12:00 西昌午饭·火盆烧烤告别餐
西昌停留1h：吃一顿火盆烧烤+充电。不亏！
🛑 返程可选停留：安顺场(红军渡·石棉下高速11km·免费·1h) 或 汉源湖(免费·拍照20min)
走走停停不那么累。在安顺场和汉源湖各停一下，缓解长途驾驶疲劳。
🏠 ~19:00 回到成都
辛苦但值得！如果实在太累，可以在石棉/雅安住一晚，第二天再回（但就需要4/7不上班了）。
