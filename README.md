## ✨ 功能特性

### 📊 方案总览页 (`index.html`)
- **16 条自驾线路**，按方向分 6 组（南线/西线/东线/东北线/特种线/周边短线）
- **多维筛选**：按方向、距离、天气过滤
- **天气实况**：集成中央气象台数据，区分晴/多云/阵雨/小雨/中雨等
- **综合排名**：风景/充电/路况/性价比 四维评分
- **速览对比表**：一表看清所有方案的核心参数

### 🏖🌊 详细攻略页 (`xichang.html` / `luguhu.html`)
- **☀️ 三天天气**：实时天气数据 + 穿衣/出行建议
- **🛤 沿途风景**：去程沿途看点推荐
- **🔌 充电规划**：纯电车分段充电策略 + 应急方案
- **🗺 景点概览**：含最佳拍照时间/光线指南/实景图片入口
- **📋 行程方案**：Tab 切换多种方案（轻松版/效率版、Plan A/B）
- **🍽 美食推荐**：每站具体店铺名 + 招牌菜 + 人均价格
- **💰 费用预估**：1人/2人 三列对比表，共摊/可选一目了然
- **⚠️ 避坑提示**：踩坑经验汇总

### 🛡 安全保障模块
- **🆘 紧急联系**：高速救援/110/120/品牌救援/保险救援/沿途医院
- **📴 离线预案**：离线地图/充电桩截图/攻略离线保存
- **🚗 山路安全**：具体险段标注 + 安全驾驶要点（泸沽湖线 800+ 弯道）
- **⚡ 充电应急**：趴窝处理 SOP + 备选充电方案
- **📡 实用信息**：信号/现金/海拔/门票/高速免费/返程拥堵

### 📋 物品清单 (`checklist.html`)
- **49 项物品**，分 7 大类（车辆/电子/生活/衣物/食物/应急/娱乐）
- **可交互勾选**，进度条实时更新
- **`localStorage` 持久化**，关闭浏览器后进度不丢失
- 一键全部重置

### 🧭 交互体验
- **页内目录导航 (TOC)**：横向滚动锚点按钮，一键跳转任意区块
- **可折叠区块**：参考性内容默认折叠，减少视觉长度
- **浮动导航**：右下角快捷按钮（回顶 ↑ / 首页 🏠 / 其他攻略 / 清单 📋）
- **平滑滚动**：点击 TOC 平滑滚动到目标区块
- **打印优化**：`@media print` 白底黑字 + 折叠区块强制展开

## 🚀 使用方式

### 方式一：直接打开
```bash
# 双击打开 index.html 即可
open index.html
```

### 方式二：本地服务器（推荐）
```bash
cd where-to-go
python3 -m http.server 8765
# 浏览器打开 http://localhost:8765
```

### 方式三：离线使用
出发前将整个 `where-to-go/` 文件夹复制到手机/平板，用浏览器打开 `index.html`。  
⚠️ 山区无信号时仍可正常使用（纯静态页面，无网络依赖）。
